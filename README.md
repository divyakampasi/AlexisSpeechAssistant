# LilySpeechAssistant
Lily is a speech assistant that recognizes voice spoken into a microphone and serves the user accordingly.

## Libraries Used 

* pip install speechrecognition
* pip install pyttsx3
* pip install pyaudio
* pip install playsound
* pip install PyObjC

## Currently used Voice Commands

What is your name?
What time is it
Search
Find Location
Exit
